## Design Decisions (all binding for execution)

### DD-2: Directory — `Assets/Scripts/Paths/`
Calculator is queried by all 3 pillars, so it lives in neutral `Paths/` (not inside `Roguelite/`).
Namespace: `TomatoFighters.Paths`.

### DD-3: 3 separate files
`FinalStats`, `StatModifierInput`, and `CharacterStatCalculator` each get their own file for discoverability.

### DD-4: float[] arrays + `StatModifierInput.Default()` factory
Modifier inputs are `float[]` indexed by `StatType` (11 values). `Default(CharacterBaseStats)` initializes neutral values (0f additive, 1.0f multiplicative). `CancelWindow` slot is zeroed and silently ignored by the calculator.

```csharp
var input = StatModifierInput.Default(characterBaseStats);
FinalStats result = calculator.Calculate(input);
```

### DD-5: `FinalStats.GetAttackForMode(AttackMode)` helper
Absorbs the `-1f` sentinel internally so Combat/World never see it:

```csharp
public float GetAttackForMode(AttackMode mode) => mode switch
{
    AttackMode.Melee     => attack,
    AttackMode.Ranged    => rangedAttack >= 0f ? rangedAttack : attack,
    AttackMode.Throwable => throwableAttack,
    _                    => attack
};
```

`AttackMode` enum (`Melee`, `Ranged`, `Throwable`) goes in `Shared/Enums/` since Combat pillar needs it.
